## 🎯 分析场景实例

### 1. 相关性分析示例

#### 分析目标
研究生活满意度各维度之间的关系强度

#### 操作步骤
1. 选择数值变量：生活满意度1、生活满意度2、生活满意度3
2. 选择相关性分析功能
3. 查看相关系数矩阵和显著性检验

#### 预期结果
```
相关系数矩阵：
                生活满意度1  生活满意度2  生活满意度3
生活满意度1        1.000      0.752      0.748
生活满意度2        0.752      1.000      0.731
生活满意度3        0.748      0.731      1.000

所有相关系数在0.01水平上显著(p < 0.001)
```

#### 结果解释
- 生活满意度各维度间存在强正相关(r > 0.7)
- 表明量表内部一致性良好
- 支持生活满意度的单一因子结构假设

### 2. 独立样本t检验示例

#### 分析目标
比较实验组与对照组在生活满意度上的差异

#### 操作步骤
1. 分组变量：实验组
2. 检验变量：生活满意度1
3. 运行独立样本t检验

#### 预期结果
```
Levene方差齐性检验：F = 0.234, p = 0.629 (方差齐性假设成立)

独立样本t检验结果：
实验组均值：4.82 ± 1.23
对照组均值：4.21 ± 1.19
t值：4.267, df = 498, p < 0.001
Cohen's d = 0.51 (中等效应量)
95%置信区间：[0.33, 0.89]
```

#### 结果解释
- 实验组生活满意度显著高于对照组
- 效应量为中等程度，具有实际意义
- 结果支持实验干预的有效性

### 3. 方差分析示例

#### 分析目标
比较不同地区居民的月收入差异

#### 操作步骤
1. 因子变量：地区
2. 因变量：月收入
3. 运行单因素方差分析

#### 预期结果
```
方差分析结果：
F(3,496) = 28.654, p < 0.001, η² = 0.148

各组均值：
北京：12,435 ± 3,246元
上海：11,987 ± 3,108元
广州：9,876 ± 2,754元
深圳：10,234 ± 2,891元

Tukey HSD事后检验：
北京 > 广州 (p < 0.001)
上海 > 广州 (p < 0.001)
北京、上海 > 深圳 (p < 0.05)
```

#### 结果解释
- 不同地区间收入差异显著
- 北京、上海收入水平最高
- 地区因素解释了约15%的收入变异

### 4. 交叉分析示例

#### 分析目标
探索性别与教育水平的关联性

#### 操作步骤
1. 行变量：性别
2. 列变量：教育水平
3. 运行交叉分析（卡方检验）

#### 预期结果
```
交叉表：
         高中    本科    硕士    博士    总计
男性      45     78      52      25     200
女性      55     89      43      13     200
总计     100    167      95      38     400

χ² = 6.847, df = 3, p = 0.077
Cramér's V = 0.131 (弱关联)
```

#### 结果解释
- 性别与教育水平间关联不显著(p > 0.05)
- 但存在一定趋势：男性在高学历组占比稍高
- 关联强度较弱，实际意义有限

## 📊 结果解释通用指南

### 统计显著性
- **p < 0.001**: 极显著，***
- **p < 0.01**: 非常显著，**
- **p < 0.05**: 显著，*
- **p ≥ 0.05**: 不显著，ns

### 效应量解释
#### 相关系数(r)
- |r| < 0.3：弱相关
- 0.3 ≤ |r| < 0.7：中等相关
- |r| ≥ 0.7：强相关

#### Cohen's d
- |d| < 0.2：小效应
- 0.2 ≤ |d| < 0.8：中等效应
- |d| ≥ 0.8：大效应

#### η²(偏eta平方)
- η² < 0.01：小效应
- 0.01 ≤ η² < 0.14：中等效应
- η² ≥ 0.14：大效应

### 模型评估指标
#### R²(决定系数)
- R² < 0.1：解释力弱
- 0.1 ≤ R² < 0.3：解释力中等
- R² ≥ 0.3：解释力强

#### 信度系数(α)
- α < 0.6：不可接受
- 0.6 ≤ α < 0.7：可疑
- 0.7 ≤ α < 0.8：可接受
- α ≥ 0.8：良好

## ⚠️ 常见误区提醒

1. **显著性 ≠ 重要性**: 统计显著不代表实际重要
2. **相关性 ≠ 因果性**: 相关关系不等于因果关系
3. **样本量影响**: 大样本容易达到统计显著
4. **多重比较**: 多次检验需要校正α水平
5. **假设检验**: 注意检验假设前提条件
