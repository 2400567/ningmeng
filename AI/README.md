# AI 数据分析系统

## 项目概述

AI 数据分析系统是一个功能强大的一站式数据分析平台，集成了数据处理、可视化分析、机器学习预测和AI助手功能。系统通过直观的界面帮助用户快速导入数据、进行数据清洗、生成多样化的可视化图表、应用机器学习模型，并自动生成专业的分析报告。

## 功能特性

### 核心功能
- **数据导入与导出**：支持CSV格式数据文件的导入和导出
- **智能数据预处理**：自动识别数据类型、处理缺失值、异常值检测
- **多样化图表可视化**：支持散点图、柱状图、折线图、直方图、热力图等多种图表类型
- **图表智能推荐**：根据数据特征自动推荐最适合的可视化图表类型
- **机器学习模型选择**：提供多种分类和回归模型，自动推荐最佳模型
- **实时进度跟踪**：直观的进度条和状态显示，优化用户体验
- **Word报告自动生成**：一键生成包含图表和专业分析总结的Word报告
- **AI助手辅助**：内置AI智能体，提供数据分析咨询和报告撰写建议

### 技术特色
- **交互式Web界面**：基于Streamlit构建的响应式用户界面
- **模块化设计**：清晰的代码结构，便于扩展和维护
- **全面的数据处理能力**：支持复杂的数据转换和特征工程
- **专业的可视化效果**：高质量的图表输出，支持自定义样式
- **集成化工作流**：从数据导入到报告生成的完整流程支持

## 安装说明

### 系统要求
- Python 3.8 或更高版本
- 足够的内存空间（推荐8GB以上）
- 支持中文显示的环境

### 安装步骤

1. **克隆或下载项目**

   ```bash
   git clone <repository_url>
   cd AI
   ```

2. **创建虚拟环境**（推荐）

   ```bash
   # Windows
   python -m venv venv
   venv\Scripts\activate
   
   # macOS/Linux
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **安装依赖包**

   ```bash
   pip install -r requirements.txt
   ```

## 使用指南

### 启动应用

在项目根目录下运行以下命令启动Streamlit应用：

```bash
streamlit run src/ui/app.py
```

应用启动后，将自动在默认浏览器中打开，或通过 http://localhost:8501 访问。

### 基本使用流程

1. **数据导入**：
   - 点击"浏览文件"按钮，选择CSV格式的数据文件
   - 系统会自动加载数据并显示数据预览

2. **数据处理**：
   - 查看数据概览信息
   - 选择数据清洗选项（处理缺失值、删除重复值等）
   - 应用数据转换操作

3. **数据分析与可视化**：
   - 查看系统推荐的图表类型
   - 选择并配置需要生成的图表
   - 调整图表参数，实时预览效果

4. **模型预测**（可选）：
   - 选择预测目标列
   - 系统自动推荐适合的机器学习模型
   - 查看模型性能评估结果

5. **报告生成**：
   - 选择需要包含在报告中的内容
   - 点击"生成Word报告"按钮
   - 下载生成的报告文件

6. **AI助手咨询**：
   - 在侧边栏中使用AI助手功能
   - 输入关于数据分析的问题
   - 获取专业的分析建议和指导

## 模块说明

### 数据处理模块
负责数据的加载、清洗、转换和特征工程，提供灵活的数据处理功能。

### 可视化模块
提供丰富的图表生成功能，支持多种图表类型，并能根据数据特征智能推荐最适合的可视化方式。

### 模型选择模块
集成多种机器学习算法，自动根据数据特征和任务类型选择最佳模型，并提供模型评估功能。

### 报告生成模块
将分析结果和图表整合到Word文档中，生成专业的分析报告。

### AI助手模块
基于规则的AI智能体，为用户提供数据分析相关的咨询服务和报告撰写建议。

### UI模块
基于Streamlit构建的交互式用户界面，连接各功能模块，提供直观的操作体验。

## 常见问题解答

### 问题：无法显示中文
**解决方案**：确保系统中安装了中文字体，或修改matplotlib的字体配置。

### 问题：数据导入失败
**解决方案**：检查CSV文件格式是否正确，确保编码为UTF-8，列名无特殊字符。

### 问题：模型预测结果不准确
**解决方案**：尝试使用数据预处理功能优化数据质量，或手动选择其他模型进行比较。

### 问题：生成的Word报告格式异常
**解决方案**：确保已安装python-docx库的最新版本，检查图表尺寸是否过大。
